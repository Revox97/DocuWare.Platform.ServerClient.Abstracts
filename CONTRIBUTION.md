# CONTRIBUTION
The following guidelines are intended to help you contribute to the DocuWare.Platform.ServerClient.Abstracts project effectively. We welcome each contribution to the project.

## Style and format
DocuWare.Platform.ServerClient.Abstracts follows standard C# coding conventions. Please ensure that your code adheres to these conventions for consistency.

## What to work on
Please prioritize items in the [issue list](https://github.com/Revox97/DocuWare.Platform.ServerClient.Abstracts/issues).

You are also welcome to work on something not on the list. Please keep in mind that the community might suggest changes or refrain from accepting an unsolicited contribution if it strays too far from the goals of this project.
## Pull requests
Submit your pull request to the [master branch](https://github.com/Revox97/DocuWare.Platform.ServerClient.Abstracts/tree/master) of the main repository. We consider any outstanding pull request ready to be merged in, so please make sure that the functionality is complete when submitting your change.
